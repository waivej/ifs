# IFS Student Galleries

Over 13 years of classes and summer programs, students created **13,498 images** across **53 galleries** — college math courses, summer CAMPY programs, and The Gilbert School, from 2004 through 2017.

Each row shows the prompt image used to inspire that class, the gallery name, and the number of student images. The prompt images were created by the course instructor.
